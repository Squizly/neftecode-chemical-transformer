### 1. Условия эксперимента и базовые параметры (10 признаков)
Эти признаки одинаковы для всех строк внутри одного `scenario_id`.

| Признак | Описание | Зачем |
| :--- | :--- | :--- |
| **`scenario_id`** | Уникальный ID эксперимента. | Для группировки строк в единый граф/сет (DeepSets). |
| **`component`** | 	Название компонента (базовое масло, присадка). | Идентификатор вещества. Используется для извлечения свойств и генерации базового Embedding. |
| **`temp`** | Температура нагрева (150-160°C). | Задает скорость теплового старения (кинетика Аррениуса). |
| **`time`** | Время испытания (168-216 часов). | Длительность термического воздействия на смесь. |
| **`biofuel`** | % добавленного биотоплива. | Мощнейший катализатор образования свободных радикалов. |
| **`catalyst`** | Доза металлического катализатора (1 или 2). | Имитирует износ стружки в двигателе, ускоряя окисление. |
| **`mass_norm`** | Доля компонента (строго от 0 до 1). | Определяет "вес" свойств данного компонента в итоговой смеси. |
| **`hidden_pct`** | Процент из названия партии (например, "3%"). | Раскрывает истинную концентрацию действующего вещества в растворе присадки. |
| **`OTHER_severity_idx`** | Индекс жесткости ($Temp \times Time$). | Учит модель понимать накопительный эффект теплового удара. |
| **`OTHER_bio_risk`** | Зона риска (150°C + биотопливо). | Явный нелинейный триггер для предсказания "взрыва" вязкости. |

### 2. Молекулярная химия (8 признаков)
Структурный паспорт молекул (сгенерирован по SMILES).

| Признак | Описание | Зачем |
| :--- | :--- | :--- |
| **`mol_wt`** | Вес одной молекулы. | Тяжелые молекулы (полимеры) сильнее загущают масло при деградации. |
| **`logp`** | Липофильность. | Показывает, выпадет ли присадка в осадок или останется в масле. |
| **`rings`** | Число колец в молекуле. | Кольца (особенно ароматика) поглощают энергию, повышая термостабильность. |
| **`cnt_Ca`, `cnt_Zn`, `cnt_S`** | Число атомов Ca, Zn, S в молекуле. | Указывает на тип присадки (Ca = моющая, Zn+S = противоизносная). |
| **`COMP_total_metals`** | Сумма всех металлов. | Показывает общую "зольность" и ресурс защитного пакета присадок. |

### 3. Лабораторный химический состав (COMP_) (19 признаков)
Анализы, показывающие, из чего состоит присадка и сколько в ней активных элементов.

| Признак | Описание | Зачем |
| :--- | :--- | :--- |
| **`COMP_Щелочное число...`** (ASTM и ГОСТ) | TBN. Запас щелочи. | Главный защитник от окисления. Нейтрализует кислоты. Если TBN высокое, масло живет долго. |
| **`COMP_Кислотное число_ГОСТ 11362`** | TAN. Стартовая кислотность. | Если высока изначально, процесс деградации запустится быстрее. |
| **`COMP_Массовая доля...`** (Ca, S, P, Zn) | Точный % металлов и неметаллов. | P и S работают как ловушки для радикалов, Ca отмывает нагар. Модель считает их истощение. |
| **`COMP_Общее содержание азота...`** (и Азот_NoMethod) | % Азота в смеси. | Азот — основа дисперсантов. Он удерживает сажу во взвеси, не давая маслу густеть. |
| **`COMP_Активный Азот / Кислород...`** | Реакционные центры молекул. | Места, которыми присадка "цепляется" за грязь в масле. |
| **`COMP_Содержание Бора...`** | % Бора. | Бор повышает термостойкость и образует защитное "стекло" на металле. |
| **`COMP_Содержание MgCO3, CaCO3...`** | Карбонатное ядро. | Это резерв щелочи внутри мицеллы детергента. |
| **`COMP_Содержание мыла / масла / воды`** | Состав раствора присадки. | Вода ускоряет разрушение (гидролиз), а "мыло" удерживает щелочь. |
| **`COMP_Содержание металла (Ca/Mg)...`** | Доля активного металла. | Дублирующий (уточняющий) признак для `COMP_Массовая доля кальция`. |
| **`COMP_Содержание насыщ. у/в...`** | % предельных углеводородов. | Чем их больше, тем масло стабильнее (меньше двойных связей для атаки кислородом). |
| **`COMP_Содержание серы, мг/кг_NoMethod`** | Микродозы серы в базе. | Признак чистоты базового масла. |

### 4. Вязкость и Реология (VISC_ и DENS_) (12 признаков)
Физическое состояние масла ДО начала теста.

| Признак | Описание| Зачем |
| :--- | :--- | :--- |
| **`VISC_Кинематическая вязкость...`** (40°C, 100°C, NoMethod) | Стартовая густота масла. | Это "Baseline". Модели нужно знать старт, чтобы предсказать % изменения. |
| **`VISC_index_ratio`** | Отношение вязкости (40/100). | Показывает, насколько сильно масло разжижается при нагреве. |
| **`VISC_Индекс вязкости_ГОСТ...`** | ГОСТовский расчет индекса. | Чем выше индекс, тем качественнее базовое масло. |
| **`VISC_Индекс стабильности...`** | Механическая прочность. | Покажет, разрушится ли полимер-загуститель еще до химического окисления. |
| **`VISC_Динамическая вязкость CCS...`** (-15°C до -35°C) | Имитация холодного пуска. | Зависит от парафинов в базе. Косвенно выдает химический состав базового масла. |
| **`DENS_Плотность при 15°С / 20°С...`** | Удельный вес. | Легкие масла (ПАО) окисляются медленнее, чем тяжелые (минеральные 1 группы). |

### 5. Энергетика и Квантовая химия (ENER_) (5 признаков)
Глубокие параметры реакционной способности молекул.

| Признак | Описание | Зачем |
| :--- | :--- | :--- |
| **`ENER_Энергия ВЗМО / НСМО`** | Энергия молекулярных орбиталей. | Показывает "готовность" молекулы принять или отдать электрон в реакции с кислородом. |
| **`ENER_Потенциал ионизации...`** | Сила удержания электрона. | Чем ниже потенциал, тем лучше присадка работает как антиоксидант (жертвует собой). |
| **`ENER_Энергия диссоциации связи Х-Н`** | Прочность связи с водородом. | Чтобы обезвредить радикал, антиоксидант отдает водород. Чем слабее эта связь, тем он эффективнее. |
| **`ENER_Химический потенциал...`** | Термодинамическая стабильность. | Общий маркер склонности вещества к химическим превращениям. |

### 6. Структурные и Коллоидные параметры (OTHER_ и TEMP_) (23 признака)

| Признак | Описание | Зачем |
| :--- | :--- | :--- |
| **`TEMP_Температура застывания / плавления`** | Точки смены фазового состояния. | Косвенный маркер длины молекул и наличия кристаллических решеток. |
| **`OTHER_Испаряемость по NOACK...`** | Угар масла (%). | При нагреве до 150°С легкие фракции улетают, из-за чего масло густеет чисто физически. Очень важный признак! |
| **`OTHER_Последовательность 1, 2, 3...`** | Тесты на пену (мл). | Пена = воздух = кислород. Масло, которое пенится, окисляется быстрее из-за площади контакта. |
| **`OTHER_Деэм.вода / масло / время / эмульсия`** | Тест на отделение воды. | Влага — враг присадок (вызывает их гидролиз). Плохая деэмульсация = быстрый выход присадок из строя. |
| **`OTHER_Определение содержания воды (Карл Фишер)`**| Точный замер влаги (ppm). | Вода запускает деградацию сложных эфиров и детергентов. |
| **`OTHER_Размер мицелл, нм`** <br> **`OTHER_Отношение Мыло/Основание`** | Структура моющей присадки. | Описывает "броню", в которой хранится запас щелочи. Чем меньше мицелла, тем выше площадь защиты. |
| **`OTHER_Масса гидрофобного хвоста...`**<br>**`OTHER_Длина углеродной цепи`** | Размеры молекулы. | Удерживают присадку в растворенном виде. Если хвост короткий — присадка выпадет в осадок. |
| **`OTHER_Разветвленность радикала`**<br>**`OTHER_Структура УВ-радикала`** | Форма молекулы. | Разветвленные молекулы хуже окисляются (стерические затруднения для атаки кислородом). |
| **`OTHER_Стерический фактор, Å3`** | Пространственный объем молекулы. | Показывает, насколько плотно молекула закрывает свой "активный центр" от врагов. |
| **`OTHER_Степень полисульфидности`** | Длина "серных мостиков". | Важно для антиизносных присадок. Длинные мостики активнее, но быстрее разрушаются при 150°C. |
| **`OTHER_Атомное отношение P:Zn`** | Баланс фосфора и цинка. | Если отношение не 1:1, значит, перед нами сложная или нестандартная противоизносная присадка. |
| **`OTHER_Дипольный момент, Д`** | Полярность. | Полярные базы (эфиры) лучше растворяют грязь, но быстрее окисляются. |
| **`OTHER_Группа по API`** | Класс базового масла (I, II, III...). | Чем выше класс, тем меньше серы и выше стабильность. |
| **`OTHER_Номер CAS...`** (2 штуки) | Регистрационные номера. | - |
| **`OTHER_Цвет_ASTM D1500`** | Оптическая прозрачность. | Темный цвет изначально указывает на низкую степень очистки базового масла. |

### 7. Целевые переменные (Targets) (3 признака)
То, что нейросеть будет учиться предсказывать.

| Признак | Описание |
| :--- | :--- |
| **`target_visc`** | Относительный прирост вязкости (%). Главная физическая проблема (масло густеет). |
| **`target_oxid`** | Степень окисления (А/см). Главная химическая проблема (масло гниет). |
| **`target_visc_log`** | Логарифм вязкости. Наша техническая колонка для правильного обучения нейросети без сбоев на выбросах. |
